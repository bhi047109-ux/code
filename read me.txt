# 5-bit Binary Encoder/Decoder

This is a simple web tool that encodes and decodes text using a custom binary system:
- A = 1 → `00001`
- B = 2 → `00010`
- ...
- Z = 26 → `11010`
- Space = 0 → `00000`

## How to Use
1. Type text (e.g., `HELLO WORLD`) and click **Encode** to get binary.
2. Paste binary (e.g., `01000 00101 ...`) and click **Decode** to get text.
